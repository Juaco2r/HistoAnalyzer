# Classifier files

Classifier JSON files are not included because they are study-specific.

Select them in the GUI:

- Tissue classifier: QuPath `OpenCVPixelClassifier` with ANN_MLP and RGB Gaussian features.
- Anthracosis classifier: QuPath `OpenCVPixelClassifier` with RTrees and RGB Gaussian features.
- DAB classifier: H-DAB stain-2 constant threshold classifier.

Validate classifier compatibility on representative images before batch analysis.
