HistoAnalyzer Windows portable edition
======================================

IMPORTANT: EXTRACT THE ENTIRE ZIP BEFORE RUNNING.

1. Close this ZIP window.
2. Right-click the downloaded ZIP file and choose "Extract All...".
3. Open the extracted HistoAnalyzer folder.
4. Run HistoAnalyzer.exe from that extracted folder.

Do not run HistoAnalyzer.exe directly inside the ZIP and do not copy the EXE
away from its _internal folder. This portable build is a PyInstaller one-folder
bundle and all files must remain together.

For the simplest installation, use HistoAnalyzer-Windows-x64-Setup.exe instead.
